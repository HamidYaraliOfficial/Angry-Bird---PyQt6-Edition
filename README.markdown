# 🇺🇸 English

## Overview

**Angry Bird - PyQt6 Edition** is a modern desktop implementation of the classic bird-launching game developed entirely with Python and PyQt6. The project features smooth animations, multiple bird types, destructible structures, particle effects, multilingual support, theme switching, and progressively challenging levels.

This project demonstrates game development concepts using the PyQt6 framework, including physics simulation, collision detection, custom painting, animation systems, and responsive user interface design.

---

## Features

### Gameplay
- Slingshot-based launching mechanics
- Physics-driven bird movement
- Multiple bird types with different strengths
- Destructible blocks and structures
- Pig health system
- Progressive level difficulty
- Dynamic scoring system
- Multiple generated levels

### Graphics & Effects
- Smooth animations
- Particle explosion effects
- Dynamic bird trails
- Day and night themes
- Responsive game rendering
- Custom character drawing using QPainter
- Camera tracking system

### User Interface
- Modern PyQt6 interface
- Real-time score tracking
- Level progression display
- Remaining birds counter
- Remaining pigs counter
- Restart functionality
- Next level functionality

### Internationalization
- English language support
- Persian (فارسی) language support
- Chinese (中文) language support

# 🇨🇳 中文

## 项目简介

**愤怒的小鸟 - PyQt6 版本** 是一个使用 Python 和 PyQt6 开发的现代桌面游戏项目。项目包含物理模拟、碰撞检测、粒子特效、多种小鸟类型、可破坏建筑、多语言支持以及关卡系统。

该项目展示了如何利用 PyQt6 构建完整的二维游戏和现代图形用户界面。
